## Bài thu hoạch “AWS Workshop: Data Science on AWS – Mở khóa sức mạnh dữ liệu cùng điện toán đám mây”

### Mục đích của sự kiện
- Giới thiệu quy trình **Data Science trên AWS**, từ thu thập, xử lý, huấn luyện đến triển khai mô hình.  
- Mang đến trải nghiệm thực hành với các dịch vụ AI/ML như **Amazon SageMaker**, **Amazon Rekognition**, **Amazon Comprehend**, và **Amazon Lex**.  
- Hiểu cách tích hợp các dịch vụ AWS để xây dựng giải pháp AI hoàn chỉnh trong thực tế.

### Nội dung nổi bật

#### Tổng quan Data Science on AWS
- Quy trình **Machine Learning Workflow**:  
  *Data Collection → Data Preparation → Feature Engineering → Model Training → Evaluation → Deployment → Monitoring*  
- **Amazon SageMaker**: Huấn luyện và triển khai mô hình tự động, bảo mật và tối ưu.  

#### Amazon Comprehend
- Dịch vụ NLP trích xuất thông tin từ văn bản: Entities, Sentiment, Topics, Syntax, Document Classification.  

#### Amazon Rekognition
- Phân tích hình ảnh/video: nhận diện khuôn mặt, vật thể, cảm xúc, văn bản.  
- Workflow: *Upload → S3 → Lambda → Rekognition → DynamoDB/Elasticsearch*.  

#### Amazon Lex
- Xây dựng chatbot thông minh sử dụng giọng nói và văn bản.  
- Tích hợp với Lambda và DynamoDB.  

#### SageMaker & AutoML
- Hỗ trợ **Scikit-learn, TensorFlow, PyTorch**, hoặc custom Docker container.  
- AutoML không cần code với **SageMaker Studio/Canvas**.  
- Triển khai nhanh bằng **API Endpoint**.
