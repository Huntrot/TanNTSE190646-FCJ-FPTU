## Event Report – Building Agentic AI – CloudThinker x AWS  
**Topic:** Context Optimization with Amazon Bedrock  
**Date:** December 5th, 2025  
**Venue:** 26th Floor, Bitexco Financial Tower  

---

### Event Objectives
- Understand the concept of **Agentic AI** and how to optimize context when building AI applications.  
- Learn how to use **Amazon Bedrock** to develop AI systems that are scalable, secure, and efficient.  
- Explore methods for designing multi-step reasoning workflows for AI agents.  
- Connect with cloud engineers, AI engineers, and solution architects in the local community.

### Key Highlights

#### 1. Overview of Agentic AI
- Introduction to **Agentic AI**, where AI systems don’t just generate text but execute tasks through well-defined workflows.  
- Importance of **context optimization** to reduce costs, improve accuracy, and maintain consistency.

#### 2. Context Optimization with Amazon Bedrock
- Use of **Knowledge Bases**, **Embeddings**, and **RAG** to supply precise context to AI models.  
- Techniques for reducing input tokens such as summarization, context compression, and layered context management.  
- Prompt design strategy for agent workflows, including:  
  - Task breakdown  
  - Step-by-step reasoning  
  - Tool calling and output safety checks  

#### 3. Real-world Agent Workflow Development
- CloudThinker shared how they build production-ready agent workflows.  
- Example agent capabilities:  
  - Customer Support automation  
  - Report generation  
  - Multi-step decision pipelines  
- Focus on **observability**, error control, and cost optimization.

#### 4. AWS Solution Architect Insights
- AWS presented best practices for running Bedrock in production environments.  
- Integration patterns using:  
  - **Bedrock Agents**  
  - **AWS Lambda**  
  - **Step Functions**  
  - **S3 + Knowledge Base**  
- Emphasis on **security guardrails**, logging, and access control.
